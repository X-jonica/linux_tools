Comment mettre à jour ou modifier le thème de GRUB ?

1️⃣ Activer l’extension “User Themes”:

* Ouvrez l’application Extensions de GNOME.
* Activez l’extension User Themes pour pouvoir appliquer des thèmes personnalisés.

2️⃣ Télécharger un thème GRUB:

* Rendez-vous sur [https://www.gnome-look.org/browse?cat=109\&ord=latest](https://www.gnome-look.org/browse?cat=109&ord=latest).
* Choisissez la version adaptée à votre langue (par exemple Archer_cn pour le français).

3️⃣ Extraire et copier le thème:

* Une fois le téléchargement terminé, décompressez le fichier.
* Ouvrez un terminal et copiez le thème dans le dossier GRUB :
  sudo mkdir -p /usr/share/grub/themes
  sudo cp -r Archer_cn /usr/share/grub/themes
* Si le dossier themes n’existe pas, la première commande le crée automatiquement.

4️⃣ Configurer GRUB pour utiliser le thème:

* Ouvrez le fichier de configuration GRUB :
  sudo nano /etc/default/grub
* Ajoutez ou modifiez la ligne suivante :
  GRUB\_THEME="/usr/share/grub/themes/Archer\_cn/theme.txt"

5️⃣ Mettre à jour GRUB:

* Exécutez la commande suivante pour appliquer le thème :
  sudo update-grub

6️⃣ Redémarrer le PC:

* Redémarrez votre ordinateur pour voir le nouveau thème GRUB appliqué.
